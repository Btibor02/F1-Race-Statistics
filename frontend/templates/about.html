About the Project

This project showcases Formula 1 race statistics using FastF1 and Flask.

Our Mission

To provide F1 fans with real-time race analytics and visualizations in a sleek and responsive web interface.

Technology Stack

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Python, Flask, FastF1
  • Database: SQLite + SQLAlchemy
  • Visualization: Chart.js
